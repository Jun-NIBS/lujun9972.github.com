查找某个文件属于哪个package的方法

目录

  • Debian/Ubuntu
  • Archlinux
  • Redhat,Fedora,Centos

从https://www.garron.me/en/linux/find-which-package-library-belongs.html学到的

Debian/Ubuntu

我们需要借助 apt-file 来实现 

sudo apt-get install apt-file -y

然后更新数据库 

apt-file update

然后搜索 

apt-file search ${file}

Archlinux

若只需要在已经安装的包内查询，则可以直接使用pacman 

pacman -Qo ${file}

若需要查询所有数据库中的包，则使用 pkgfile 

sudo pacman -S pkgfile --noconfirm

更新数据库信息 

sudo pkgfile --update

使用pkgfile查询 

pkgfile ${file}

Redhat,Fedora,Centos

在已安装的包中检查执行 

rpm -qf $(which ${file})

或者，在所有包中检查，执行 

yum provides */${file}
