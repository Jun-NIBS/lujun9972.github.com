* 启用MTP支持

要让archlinux支持MTP协议，需要安装 =mtpfs=
#+BEGIN_SRC shell :dir /sudo:: :results org
  sudo pacman -S mtpfs --noconfirm
#+END_SRC

但对于Android 4+的设备，光有 =mtpfs= 还不够，还需要安装 =jmtpfs=, 而 =jmtpfs= 是在AUR中的：
#+BEGIN_SRC shell
  yaourt -S jmtpfs
#+END_SRC
